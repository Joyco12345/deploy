# Сбор информации из интернета

Проект позволяет по ключевому слову получать список URL, скачивать страницы через сервер с отображением прогресса и сохранять их в LocalStorage браузера.

## Локальный запуск

1. Установить Node.js (версия 14 или выше).
2. Клонировать репозиторий:  
`git clone https://github.com/Joyco12345/deploy`
3. Перейти в папку проекта и установить зависимости:  
`npm install`
4. Запустить сервер:  
`npm start`  
(или `npm run razrabotka` для обновления)
5. Открыть браузер и перейти по адресу:  
`http://localhost:3000`

## Публичный доступ

Приложение развёрнуто на платформе Render и доступно по ссылке:  
**https://deploy-63lj.onrender.com**



Использование

1. Ввести ключевое слово (например: `javascript`, `python`, `html`).
2. Нажать **«Найти»**. Появится список доступных URL.
3. Выбрать один URL и нажать **«Скачать выбранный URL»**.
4. Наблюдать за прогрессом загрузки на полосе и в текстовом описании.
5. После завершения загрузки страница автоматически сохранится.
6. В разделе **«Сохранённые страницы»** выбрать любую и нажать **«Просмотреть»**, чтобы увидеть её содержимое.
7. Ненужные страницы можно удалить кнопкой **«Удалить выбранное»**.
